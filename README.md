# spellChecker
Input box that provides word suggestions using Compact Prefix Trees

![app view](input/appViewGif.gif)

## Install

```shell
$ git clone https://github.com/nishmeht7/spellChecker.git
```

## Start Server

```shell
$ mvn spring-boot:run
```

## Launch React app

```shell
$ cd webapp
$ npm install
$ npm start
```

## Navigate to

[http://localhost:3000](http://localhost:3000)
